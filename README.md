# My Profile Card

A simple and stylish profile card built using HTML, CSS, and JavaScript. This project showcases a responsive personal profile card with modern styling and interactive elements.

## Features

* Responsive design for mobile and desktop
* Customizable profile image, name, and description
* Stylish hobby buttons and social links
* Interactive elements powered by JavaScript
* Easy to edit and extend

## Getting Started

1. Clone or download this repository.
2. Open the `index.html` file in your browser.

## File Structure

```bash
project-folder/
│
├── index.html      # Main HTML structure
├── style.css       # Styling for the profile card
├── script.js       # JavaScript functionality
└── image/          # Images and assets
```

## Customization

* Replace the profile image inside the `image` folder.
* Edit your name, profession, and personal details in `index.html`.
* Modify colors, fonts, and layout in `style.css`.
* Add more interactive features in `script.js`.

* HTML5
